Why Choose Losdol?

  • Easier to configure: Pre-configured and ready to use.
  • Built-in Software:
    • Xorg-7.7 (with 3D support)
    • WiFi & Alsa drivers
    • Fluxbox Window Manager
    • Firefox Nightly, XFE File Explorer, and more
  • Easy for beginners: Lightweight yet user-friendly, perfect for new Linux users.

Losdol is a distribution based on Tiny Core Linux. It retains the lightweight and modular design of Tiny Core Linux, while pre-configuring popular software and tools to provide an out-of-the-box experience for users.

Installation Guide

Follow the Tiny Core Linux Installation Guide with the following Losdol-specific settings:

  1. Path to core.gz: /mnt/sr0/casper/core.gz
  2. Bootcodes: cde + any other bootcodes you need
  3. CDE directory: /mnt/sr0/cde
